SY8

Posted on Sun, Sep 10, 2023 Streamlit Yolov8 Growing

Introduction

Streamlit+Yolov8 (SY8) provides a user-friendly interactive interface based on Streamlit for the YOLOv8 object detection model. I would like to use this project to summarize and share my current skills and expertise, as well as the development process and machine vision knowledge involved, through a series of blog posts.

This application utilizes the object detection model to develop intelligent surveillance applications, such as regions of interest, electronic fences, vehicle flow statistics, and pedestrian counting. I plan to divide this project into 5 phases of development:

  1. Environment setup: Setting up a Docker environment for developing the Yolov8 model and Streamlit frontend interface.
  2. Object monitoring module: Users can define components called "Context" on the monitoring screen, and the objects detected by the object detection model are referred to as "Objects". Users can define the type, shape, and position of the Context as well as the type of Objects.
  3. Conditional logic module: Users can define the interaction logic between the Objects detected by the object detection model and the Context defined by the user.
  4. Event triggering module: When the conditional logic is satisfied, users can define the desired feedback, such as screen presentation or email alerts.
  5. UI design: All of the above modules can be operated and implemented by users on the UI interface.

I will use 5 labels to identify these phases, namely Base, Context, Condition, Event, and Final.

Problem

Based on my personal experience, conventional intelligent surveillance systems require independent application development for different scenarios using object detection algorithms, such as vehicle flow statistics, electronic fences, etc.

The key to quickly deploying intelligent surveillance applications is to develop a framework that allows customization.

Proposal

SY8 uses Streamlit as the frontend framework to design the UI interface and combines it with the latest Yolov8 object detection algorithm to develop a customizable and quickly deployable intelligent surveillance system.

Installation

Install dependencies

Use the following command to install the required dependencies:

pip install ultralytics
pip install streamlit
# or
pip install -r requirements.txt

Running on Docker

It is recommended to use the Dev Containers feature of Visual Studio Code to run the project:

  1. Press Ctrl + Shift + P
  2. Select "Dev Containers: Reopen on Container"
  3. Done! VS Code will run a container based on the Dockerfile in the project root directory

Alternatively, you can build the Docker image yourself:

docker build -t ${imagename} .

Then, run the streamlit command inside the container:

streamlit run app.py

Roadmap

Visualize model inference results

Define geometric shapes on the screen, such as points, lines, and areas
Define conditional logic between geometric shapes and target bounding boxes
Define actions triggered when conditions are met

Some Thoughts

This project provides a frontend interface for users to define their own intelligent surveillance applications. It allows users to configure their "intelligent" surveillance applications using a natural language-like expression. However, this is far from true intelligence. The challenge lies in how to achieve true natural language command for the intelligent surveillance system. For example, if a user gives the command "I don't want anyone to enter region A in the frame," the system should understand the objective and provide the desired feedback. How to make it understand natural language and combine it with system functionality is an interesting and thought-provoking problem.
